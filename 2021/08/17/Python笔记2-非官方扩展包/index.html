Python 扩展包的非官方 Windows 二进制文件

Python作为一种万能的胶水语言，依赖于各种各样的扩展包，快速找到包安装好就是一件重要的事，省时省力。

通常用pip在线安装就可以搞定

但有的时候也会失败，这时候就需要自己下载二进制包进行安装

  • 先弄清楚自己的Python是什么版本的，以方便后续下载合适的.whl文件。

win+R进入命令运行窗口，输入cmd打开命令提示符，接着输入python即可

  • 选择需要的.whl文件下载，一般选择最后的，感觉意思是最近更新的包，以下是.whl文件下载链接地址：

    Python扩展包的非官方Windows二进制文件
